Essentia + Docker: Music Interpretation Guide

Analyze and translate your music for AI companions & creative collaboration

Step 1: Prepare Your Machine

  • Install Docker Desktop with WSL2
  • Create a folder on Windows, e.g. M:\Essentia\tracks
  • Put your .mp3 or .wav files inside it

Step 2: Pull the Docker Image

docker pull ghcr.io/mtg/essentia:latest

Step 3: Create Your .bat Script

@echo off
echo.
echo Enter the filename of the track (e.g., MySong.mp3):
set /p FILE=

docker run --rm -v M:/Essentia/tracks:/music ghcr.io/mtg/essentia:latest bash -c "essentia_streaming_extractor_music '/music/%FILE%' '/music/%FILE:~0,-4%_analysis.json'"

echo.
echo Analysis complete! Check the JSON file in your tracks folder.
pause

Save it as analyze_song.bat. Double-click to run.
